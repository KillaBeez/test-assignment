Задание №4

Сверстать стили для кастомных кнопок как показано на приложенном изображении.

Стили должны применяться только к элементам у которых есть класс b-button. Длина изображения должна зависеть от длины текста (никаких фиксированных размеров). При наведении на кнопку ободок должен с плавной анимацией становиться темнее. При уведении мышки ободок так же с плавной анимацией должен возвращаться в изначальное состояние.

На изображении в верхнем ряду указан реальный размер кнопки и как она выглядит на разных фонах. Ниже увеличенный размер. Сетка на последнем изображении равна 8px.

Стиль шрифта не важен. Постараться реализовать без использования javascript и изображений. Результат должен находится в подключенном файле b-button.css
